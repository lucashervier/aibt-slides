Data Exploration

During this second class you will get comfortable with the exploratory phase of a Machine Learning project. In order to do so, we will dive a little bit more on data characterization and define properly a dataset. In addition, you will set your first machine learning pipelines especially data pre-processing. Finally, you will cover how to gain insight on your data with some analysis and visualization tools.
